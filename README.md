# 📋 Code-Spisok — Менеджер Списков в Терминале

Интерактивная программа для управления списками через командную строку. Разработана на Python с полным покрытием unit- и системными тестами.

---

## 🎯 О Проекте

**Code-Spisok** — это CLI-приложение для работы со списками через терминал. Проект создан в образовательных целях для отработки:
- 🐍 Python (работа со списками, типизация данных)
- 🧪 Автоматизированного тестирования (pytest)
- 💻 Взаимодействия с пользователем через консоль
- 📊 Управления логикой приложения

> ⚠️ Проект активно развивается и используется как площадка для внедрения новых функций и тестирования

---

## ✨ Функциональность

Программа предоставляет 15+ команд для управления списками:

| Команда | Описание |
|---------|---------|
| `add` | ➕ Добавить элемент в список |
| `ren` | 🏷️ Переименовать список |
| `rev` | 🔄 Развернуть список (reverse) |
| `show` | 👁️ Показать содержимое списка |
| `clear` | 🧹 Очистить список |
| `el_check` | 🔍 Показать все элементы с типами данных и индексами |
| `rem_el` | ❌ Удалить элемент по значению |
| `rem_ind` | ❌ Удалить элемент по индексу |
| `rep_el` | 🔄 Заменить элемент по значению |
| `rep_ind` | 🔄 Заменить элемент по индексу |
| `fd_el` | 🔎 Найти элемент по индексу |
| `fd_ind` | 🔎 Найти индекс элемента |
| `help` | ❓ Показать справку |
| `exit` | 🚪 Выход из программы |

### Поддерживаемые типы данных
- 📝 Строки
- 🔢 Целые числа (int)
- 🎯 Числа с плавающей запятой (float)
- 🎛️ Автоматическое преобразование типов при вводе

## 🧪 Тестирование

Проект включает полное покрытие автоматическими тестами:

### Запуск всех тестов

```bash
cd Code-spisok
pytest
```

### Запуск конкретного типа тестов

```bash
# Unit-тесты
pytest tests/unit/

# Системные тесты
pytest tests/system/

# E2E тесты
pytest tests/system/test_end_to_end.py

# С подробным выводом
pytest -v

# С покрытием (требует pytest-cov)
pytest --cov=src
```

### Типы тестов в проекте

- **Unit-тесты** (`tests/unit/`) — тестирование отдельных функций (валидация)
- **Системные тесты** (`tests/system/`) — тестирование команд через CLI
- **E2E тесты** — полные сценарии использования программы
